EPCFTime_t EPCFPwmCfg_t::resolution

Specifies the resolution for the pwm. Resolution refers to the mimimum time which is elapsed after every clock cycle. You can define maximum period of pwm by setting appropriate resolution value. Also,frequency of the pwm device is inverse of the pwm period. Pwm period = resolution x (2^pwmWidth). Pwm frequency = 1/Pwm period For example: Consider an 8-bit pwm device with resolution = 0.25 microseconds. Then,the maximum period of the pwm device is equal to 0.25 x 256 = 64 microseconds. To set 0.25 microseconds as the resolution, set time unit to enEPCFTimeUnit_ns and set 250 as the count. Maximum Pwm frequency = 1/64 microseconds = 15.6khz Refer to platform specific header file for supported values of resolution. Refer to "../../common/types.h" header file for the definition of EPCFTime_t structure.
