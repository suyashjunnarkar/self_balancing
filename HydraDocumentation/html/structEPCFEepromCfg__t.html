Detailed Description

Eeprom configuration structure.

Example to configure eeprom device:

EPCFEepromCfg_t eepromConfig = { enEPCFEepromDataBits_8, // 8-bits eeprom read/write 0 // Page management not supported enEPCFBooleanFalse, // Disable eeprom interrupt 0, // Not required in polling mode 0, // Not required in polling mode NULL // No hardware specific config };
