This function is used to set the compare time. Compare channel can be used to trigger an event after the specified time. A compare action and an interrupt can be associated to that event. A user callback is called if there is a callback attached for this interrupt source. The maximum value that should be passed to this function is equal to the period of the timer. Timer period = resolution x (2$^\wedge$timer\+Width). For example\+: Consider an 8-\/bit timer with resolution = 0.\+25 microseconds and configured as a periodic type. If time count == 5 and unit is equal to en\+E\+P\+C\+F\+Time\+Unit\+\_\+us, then user callback will be called after every 5 microseconds.
